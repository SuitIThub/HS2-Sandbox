# Anim Browser — getting started

![Select character and apply animation from the grid](images/anim-browser/ab-04-first-apply.gif)

> **Luna Clark** in T-pose in the viewport: **Character → Walking & Running** → click **Walking 1** (stick-figure hover preview on the card) — she begins walking.

## Basic workflow

1. In Studio workspace, **select the character(s)** to animate
2. Open Anim Browser from the **anim icon** (left toolbar)
3. Click a **sub-category** on the left
4. **Click an animation** — applied to all selected characters

## If nothing happens

- No character selected in Studio
- Selected object is prop/accessory (ignored)
- Check character summary in top bar

## First-time tips

- Groups start **collapsed** — expand with **►**
- Press **↻** after installing animation mods
- Use **Grid** for previews, **List** for fast name scanning
- Open **Help** for in-game quick reference + HS2Wiki links

## HS2 hover preview

On HS2, hover grid cards (or list rows) for an optional **stick-figure preview** — no scene character needed. Toggle and camera settings: **Options → Hover animation preview**. See [Thumbnails](Thumbnails) for hover vs captured PNGs.

## Thumbnails

Use **Capture thumbnail(s)…** in the content bar or **Options → Thumbnails** to save static PNGs for cards. Useful on KKS/KK where hover preview is unavailable.
