# Anim Browser — playback controls

Open **Controls** from top bar. Shows content when selected character has an animation.

## Controls

| Control | Effect |
|---------|--------|
| **Speed** | Slider or typed value |
| **Pause / Play** | Freeze/resume |
| **Time** | Scrub timeline |
| **Force loop** | Repeat one-shot clips |
| **Restart animation** | Replay from start |
| **Restart all in scene** | Replay every applied animation |
| **Show items** | List loaded animations on character |

## Docked vs floating

| Mode | Behavior |
|------|----------|
| **Docked** | Beside main window; closes with Anim Browser |
| **Floating** | Independent window — usable when main window closed |

Hotkey: **HotkeyToggleUndockedControls** (Configuration Manager).

## Group by proximity

**Group controls by proximity** (Options): merge control boxes only for characters within a small world-space radius.
