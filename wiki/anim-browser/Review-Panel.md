# Anim Browser — review panel

Opens for every **group** or **merge** operation. Nothing saved until **Confirm**; **Cancel** discards all staged changes.

![Merge review panel with gender and phase assignments](images/anim-browser/ab-13-review-panel.png)

> **Review add to merge: Service** — section **Female Service 2 / Behind Handjob (6 groups)** with cards like **About To Begin** and **Fast Loop**; each row has **f** / **m** gender, **—** phase, and **Skip**; **Save as** rename fields and **Confirm** / **Cancel** at the bottom.

## Controls

| Control | Effect |
|---------|--------|
| **Gender** (m/f/m2…) | Set participant role |
| **Phase** (in/loop/out/—) | Set sequence phase |
| **Skip** | Keep one animation at original category |
| **Skip all** | No card for this proposal |
| **As singles** | Show individually in merged category, no card |
| **Restore group** | Undo Skip all / As singles |
| **Confirm** | Apply entire review |
| **Cancel** | Discard — no changes |

## Sections

Proposed cards grouped under headings by original sub-category. Collapse headings for long lists.

## Empty review

No auto-bundled cards — merge/join still valid. Click **Confirm** for combined categories without grouped cards.

When joining male + female sub-categories that can pair, review may show cards after join applies.
